LibreOffice uses XHP files to create online help pages. It is an XML format that originally defined pages for the in-application help system. When they moved to online help they kept the format and developed methods to render pages as HTML. This lets them use the existing help pages, but complicates the life of new writers. You need to know the format before you can write online help content.
